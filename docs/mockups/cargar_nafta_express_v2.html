🎓 Principios de experto aplicados

1 dato obligatorio
Solo el monto es requerido. Litros, fecha, todo lo demás tiene default o es opcional → menos fricción.
Jerarquía visual
El monto es el héroe (grande, centrado). El $/L es un resultado calculado, no un input que compite.
Recognition > recall
Piloto = el usuario logueado (auto). Fecha = "Hoy, ahora". El usuario reconoce y confirma, no tipea de cero.
Progressive disclosure
90% del uso son 2 datos. Lo avanzado (Factura A, foto, odómetro) a un toque, con badge si está activo.
Ley de Fitts
CTA grande y fijo en la zona del pulgar, siempre visible. Touch targets ≥ 44px.
Feedback inmediato
El precio/L se recalcula solo. El botón muestra el monto ("Registrar · $79.995") para confirmar antes de tocar.
Prevención de error
El monto se formatea como AR ($79.995) mientras tipeás → evita el bug de decimales/miles que ya tuvimos.
Tanque lleno claro
Control propio (no un chip suelto), con micro-explicación de para qué sirve.
